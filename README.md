# 🏠 House Price Prediction Dashboard

## 📌 Project Overview
This project is a Machine Learning-based web application that predicts house prices based on user inputs such as area, number of bedrooms, bathrooms, stories, and parking spaces. The application is built using Streamlit for an interactive dashboard experience.

---

## ❗ Problem Statement
Real estate pricing is often complex and influenced by multiple factors. The goal of this project is to build a predictive model that estimates house prices accurately using historical data, helping users make better buying or selling decisions.

---

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
- NumPy

---

## 🏗️ Project Architecture
1. Data Collection (dataset.csv)
2. Data Preprocessing using Pandas
3. Model Training using Linear Regression
4. Model Prediction
5. Streamlit Web App Interface
6. User Input → Model → Price Prediction Output
